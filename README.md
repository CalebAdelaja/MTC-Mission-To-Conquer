# MTC - Mission To Conquer

## Overview

**MTC (Mission To Conquer)** is a community management application designed to help organize members, leaders, and events within the organization.

The application serves as a central platform where members can view leadership structures, register themselves, discover their sponsors and uplines, and stay updated with upcoming events.

---

## Purpose

The goal of MTC is to:

* Organize members according to their ranks/status.
* Help newbies connect with experienced members (Pros).
* Maintain a structured record of sponsors and uplines.
* Promote organizational events and activities.
* Provide a simple and accessible directory for all members.

---

## Member Categories

Members are grouped according to their status:

### Pros

* Distributor
* Manager
* Senior Manager
* Executive Manager

### Newbies

* Newbie

Each category has its own list within the application.

---

## Features

### 1. Member Directory

The application displays members in organized grid lists.

#### Pro Categories

* Distributor List
* Manager List
* Senior Manager List
* Executive Manager List

#### Newbie Category

* Newbie List

Each member card displays:

* Profile Picture
* Full Name
* Status
* Sponsor Name
* Upline Name

---

### 2. Member Registration

New members can add themselves to the application.

During registration, users provide:

* Picture
* Full Name
* Status
* Sponsor Name
* Upline Name

The system automatically places members into the correct category based on their selected status.

#### Example

If a user selects:

* Distributor → Added to Distributor List
* Manager → Added to Manager List
* Senior Manager → Added to Senior Manager List
* Executive Manager → Added to Executive Manager List
* Newbie → Added to Newbie List

---

### 3. Member Filtering

A dropdown menu allows users to filter and view members by status.

Available options:

* All Members
* Distributor
* Manager
* Senior Manager
* Executive Manager
* Newbie

This makes it easy to locate specific groups of members.

---

### 4. Events Management

The application includes an Events section where upcoming organizational activities can be displayed.

Examples:

* Monthly Cheque Rally
* Training Sessions
* Conferences
* Special Meetings

Each event contains:

* Event Flyer/Image
* Event Date
* Venue
* Description

---

### 5. Administration

Administrators are responsible for managing the platform.

Possible responsibilities include:

* Approving member registrations
* Adding and editing events
* Updating member information
* Removing incorrect entries
* Managing organizational records

Future versions may include role-based access where only authorized leaders can create events or manage members.
